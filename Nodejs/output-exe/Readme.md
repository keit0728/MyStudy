# output-exe
[nexe](https://www.npmjs.com/package/nexe)でexe化する方法をまとめます。

# DEMO
CLI（例:Windowsなら`cmd.exe`）上で下記を実行。

```console
nexe index.js
```

`output-exe.exe`が作成されます。
`cmd.exe`で実行すると下記の通り表示されるはずです。

```console
Hello Nodejs!!!
```
